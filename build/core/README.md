# @andrewjypark/subatomic-design-tokens-core

Core (base) CSS design tokens for the Subatomic design system, with no flavor applied.

## Install

```bash
npm install @andrewjypark/subatomic-design-tokens-core
```

## Usage

Import the CSS custom properties into your stylesheet:

```css
@import "@andrewjypark/subatomic-design-tokens-core/css/_variables.css";
@import "@andrewjypark/subatomic-design-tokens-core/css/abstracts.css";
```

- `css/_variables.css` — generated design tokens (colors, spacing, typography, etc.)
- `css/abstracts.css` — base/breakpoint variables shared across all flavors
