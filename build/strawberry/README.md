# @andrewjypark/subatomic-design-tokens-strawberry

Strawberry theme CSS design tokens for the Subatomic design system.

## Install

```bash
npm install @andrewjypark/subatomic-design-tokens-strawberry
```

## Usage

Import the CSS custom properties into your stylesheet:

```css
@import "@andrewjypark/subatomic-design-tokens-strawberry/css/_variables.css";
@import "@andrewjypark/subatomic-design-tokens-strawberry/css/abstracts.css";
```

- `css/_variables.css` — generated design tokens (colors, spacing, typography, etc.)
- `css/abstracts.css` — base/breakpoint variables shared across all flavors
